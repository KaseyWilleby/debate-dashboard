# **App Name**: WorkSessionScheduler

## Core Features:

- User Authentication: Allow users to log in and log out securely, managing access based on roles (admin, varsity, novice).
- Session Management: Enable users to create, edit, book, and cancel sessions with date, time, and description, and to assign sessions to host and client
- Session Messaging: Enable real-time messaging between session participants.
- Role-based Access Control: Control access to features based on user roles (admin, varsity, novice).
- Session Filtering: Allow admins to filter sessions by date, host, client, and status.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey professionalism and trust.
- Background color: Light gray (#F5F5F5) for a clean and modern feel.
- Accent color: Purple (#9C27B0) to highlight important actions and elements.
- Headline font: 'Space Grotesk' sans-serif for a modern and technical look.
- Body font: 'Inter' sans-serif for clear, readable text.
- Use 'lucide-react' icons for a consistent and clean interface.
- Use a grid layout to organize session cards and form fields.
- Subtle transitions for button hovers and modal appearances to improve user experience.
